## Form 5472 — Foreign-Owned Corporation (THE BIGGEST TIME SINK)

Form 5472 is required for each 25% foreign-owned corporation, one per related foreign party. This form caused the most issues on the first 1120 return.

### Parts to Complete

- **Part I** — Reporting Corporation info (may auto-populate from Screen 1)
- **Part III** — Related Party info: name, address, country, U.S. identifying number, FTIN, principal business activity code, relationship description, reference ID
- **Part IV** — Monetary transactions: purchases, sales, rents, commissions, loans, interest, etc.
- **Part VII** — Additional information (~8 yes/no questions)

### Critical: 8b U.S. Identifying Number

For foreign entities without a U.S. EIN, enter **`FOREIGNUS`** in the 8b field. Click precisely inside the input box — the click target is small and offset from the label.

### Critical: Part VII Checkboxes — USE HEADS DOWN ENTRY

The Yes/No checkboxes on Part VII **CANNOT be clicked directly** with computer-use mouse clicks. No coordinates work. This consumed 60+ minutes of failed attempts before discovering the fix.

**The fix: Heads Down Entry mode (Ctrl+N)**

1. Right-click → "Heads Down Entry" (or Ctrl+N)
2. Every field gets a number
3. Type field number + Enter to navigate
4. Click value input, type **X** + Enter to check
5. Escape to exit

### Part VII Field Numbers in Heads Down Entry

|----------|-------------|------------|

## LOSS Screen — NOL Carryforward

**Navigation:** Search "LOSS" or access from Screen J.

The LOSS screen ("Loss & Contribution Carryovers from Prior Years") is where NOL carryforwards from prior years are entered.

### How to Determine NOL Amounts

1. Check PY return's **Statement 4** (NET OPERATING LOSS DEDUCTION) — lists each year's loss sustained and amount available
2. Check PY **Two-Year Comparison** page, "TAXABLE INCOME BEFORE NOL DEDUCTION" line — if negative, the PY itself generated a new NOL
3. Enter ALL amounts as **positive numbers**, one per origination year

### Entry Workflow

- Click on "Unused Amount" field for the correct year row
- Type amount (positive, no commas)
- Press Tab
- Repeat for each year

### Example (Atamin International)

| Year | NOL Amount |
|------|-----------|
| 2024 | 493,231 |
| 2023 | 212,458 |
| 2022 | 769,035 |
| 2021 | 576,176 |
| 2020 | 4,154 |
| **Total** | **2,055,054** |

## Common 1120 EF Errors

| Code | Issue | Fix |
|------|-------|-----|
| 0136 | 5472 Part III incomplete | Complete all related party fields: name, address, country, business code, ref ID |
| 0168 | 5472 Part VII not answered | Use Heads Down Entry (Ctrl+N) to check all Yes/No answers |
| 0137 (CA) | CA requires 5472 PDF attachment | Print 5472 to PDF, attach via EF > PDF Attachments (user must do manually) |
