# Drake Tax 1065 — Partnership Return

Based on completing 2 Nautilus Investments LLC returns with full retrospective capture (Jeffery, Maple) — rental real estate partnerships, 2 partners each, Illinois state filing. Nautilus SSD, Cornell, and 7220 exist in the CSM as related entities but did not have learnings captured in `retro-log.md`; any work on them should follow the patterns documented here and produce its own retro entry.

**Target time: 10 minutes per return.**

**Navigation policy: use Heads Down Entry (Ctrl+N) as the default mode on every screen** — Schedule B's ~20 yes/no questions, K-1 Country of Citizenship, M-1 Tax-Basis, 8825 QBI dropdown, and Screen 2 State boxes all batch cleanly in HDE. See SKILL.md "Heads Down Entry (HDE) — PRIMARY navigation mode" for the click-(548,87)/click-(585,91) batch pattern. The 1065 learnings below were captured before HDE was promoted to primary mode; prefer HDE anywhere the older text describes direct clicking.

## Critical Rules

### Rule 1: Additional Other Expenses goes on Line 17, NOT Guaranteed Payments
On Form 8825, the detail worksheet for miscellaneous expenses (pest control, office supplies, cleaning, etc.) MUST be attached to **"Additional Other Expenses" on line 17**. Never attach it to Guaranteed Payments on line 16. This mistake cost 40 minutes on the first return.

### Rule 2: Every 4562 asset needs "For: 8825"
The "For:" field on each depreciation asset tells Drake which form receives the depreciation. For rental real estate, it must be `8825`. If left blank, depreciation won't flow to Form 8825 line 14 and income will be overstated.

### Rule 3: Fill ALL blue fields on first pass
Common blue fields people miss: K-1 Country of Citizenship → US, K-1 Max percentage owned, PIN SSN, M-1 Tax-Basis checkbox, 8825 QBI "trade or business" → Y.

### Rule 4: Never restart from scratch
If you find errors after calculating, fix them in place. Restarting wastes all prior work. Every error in Drake can be fixed by navigating to the right screen.

### Rule 5: One calculate cycle
If rules 1-4 are followed, you should only need to calculate once.

### Form 8825 — Rental Real Estate

**Navigation:** Click "8825" in screen selector.

**Property header:** Type (1=Single Family, 2=Multi-family), address, fair rental days, personal use days.

**QBI dropdown at bottom:** Set "This activity is a trade or business" to **"Y"** — this is a yellow-highlighted required dropdown. Easy to miss by scrolling past.

**Income/Expense lines:**
| Line | Field | Entry |
|------|-------|-------|
| 2a | Gross rents | Direct entry |
| 5 | Commissions | Direct entry |
| 6 | Insurance | Direct entry |
| 7 | Legal fees | Direct entry |
| 8 | Interest - mortgage | Direct entry |
| 10 | Repairs | Direct entry |
| 11 | Taxes | Direct entry |
| 12 | Utilities | Direct entry |
| 14 | Depreciation | AUTO from 4562 — do NOT enter |
| 17 | **Additional Other Expenses** | **DETAIL WORKSHEET** |

**Detail Worksheet for line 17:**
1. Double-click the Additional Other Expenses amount field
2. Click first Description cell, type name, Tab to Amount, type number
3. Repeat for each expense (pest control, supplies, management fee, etc.)
4. Exit via dropdown → "Exit"

### Schedule B — Questions

~20 yes/no questions. Typically all "No" for simple rental partnerships. Batch all checkboxes in one `computer_batch` call. Take screenshot to verify.

### M-1 — Reconciliation

**CRITICAL:** Check **"Tax-Basis"** checkbox for reporting capital accounts. Missing = EF error 1327.

### Schedule K — Credits

Check **Item 16a: "Schedules K-2 and K-3 are not required"** for domestic partnerships with no foreign activity.

### SEC — Filing Security

Enter **prior year income** from Form 8879-PE line 4. Often negative for rental partnerships.

## Common 1065 EF Errors

| Code | Issue | Fix |
|------|-------|-----|
| 0012 | Missing Country on B-1 | K-1: Country = US for each 50%+ partner |
| 0209 | Missing prior year income on SEC | Enter from Comparison Schedule |
| 0332 | Missing PIN info | Complete all PIN screen fields |
| 0430 | K-2/K-3 question | Check item 16a on K Credits tab |
| 1327 | Missing capital account basis | Check Tax-Basis on M-1 |
| 1501 | Incomplete e-file auth | Complete PIN screen |
| REQUIRED | QBI dropdown blank | Set "Trade or business" to Y on 8825 |
| IL 0010 | IL books/records location | Fill BOTH sections on Screen 2 State tab |
| IL 0174 | IL K-1s don't match | Create ILK1 entries in IL package |
| IL 9320 | IL SSN/auth missing | Add SSN on PIN + check IL Authorization |
